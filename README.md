# Tortle

This is a small game project I've been working on, I have no set goal with it yet.

# Controls

SPACE - switch between mouse and keys style of control

W - Move forwards

S - Move backwards

A - Turn left

D - Turn right

~ (Tilde) - Open command console

Esc - Exit game
